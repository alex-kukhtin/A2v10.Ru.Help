Платформа

Система A2v10 реализована с использованием технологического стека .NET Framework корпорации Microsoft.

В качестве СУБД используется Microsoft SQL-Server либо облачный сервис Azure SQL Database (service).

Серверная часть системы реализована как web-приложение на платформе Microsoft ASP.NET.MVC. Поддерживается развертывание системы в облачном сервисе Microsoft Azure как Web App Service.

Для реализации клиентской части web-приложения используется JavaScript фреймворк с открытым исходным кодом Vue.js. Клиентская часть реализована как SPA-приложение (Single Page Application). Поддерживаются все современные браузеры (Edge, Chrome, FireFox, Safari). Браузер Internet Explorer НЕ поддерживается.

Для построения отчетов и печатных форм применяется отчетная система Stimulsoft Reports.Web.

Обратите внимание! Библиотеки компании Stimulsoft НЕ входят в состав платформы. Для работы с отчетами требуется приобрести отдельную лицензию.
